Download Landsat bulk metadata XML from: http://landsat.usgs.gov/consumer.php
Select WRS Path: 39 to 40
Select WRS Row: 32 to 33
Select Dataset: "Pre-Collection: Landsat 4-8 Combined"
Select Starting: 07/01/1982
Select Ending: 09/30/2016 (or later)
Select Scene Cloud Cover: "All"
Select Output Format: "Custom XML"
Save as "metadata.xml" in the quicklooks folder
